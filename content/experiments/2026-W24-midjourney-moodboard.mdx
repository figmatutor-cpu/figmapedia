## 실험 배경

신규 클라이언트 첫 미팅에서 "분위기 예시 3개만 보여달라"고 요청받는 경우가
많습니다. 기존 워크플로우는 Pinterest 검색 → 스크린샷 → Figma에 배치로
1~2시간이 걸렸는데, Midjourney v7이 나오면서 진짜 15분이면 끝낼 수 있는지
실측해봤습니다.

## 시나리오

- **클라이언트**: 가상의 한국 SaaS 스타트업 (B2B 협업 툴)
- **요청**: "신뢰감 있으면서도 답답하지 않은 분위기" 3안
- **제약**: 30분 안에 미팅에 들고 들어가야 함

## 워크플로우

### 1단계: 키워드 추출 (3분)

브랜드 키워드를 ChatGPT로 먼저 정리 — "신뢰 + 답답하지 않음 + B2B 협업"을
시각 언어로 번역하면:

- 직선 + 라운드 코너의 균형
- 차분한 블루 톤 + 따뜻한 액센트 (오프화이트, 베이지)
- 인물 + 도구 + 자연 광선

이걸 그대로 Midjourney 프롬프트의 베이스로 사용.

### 2단계: 1차 생성 (5분)

```text
A modern minimalist workspace with collaborative tools,
soft daylight, calm blue and warm beige palette,
people working together, rounded UI elements visible on screens,
editorial photography style --ar 16:9 --v 7 --s 250
```

→ 4컷 결과 중 2컷이 즉시 사용 가능. 1컷은 인물 손가락 깨짐(여전한 약점).
1컷은 톤이 너무 차가워서 reject.

### 3단계: 변형 + 큐레이션 (5분)

가장 좋은 1컷을 시드로 `--seed`로 고정하고, 키워드만 조금씩 바꿔서 변형:

- "warm sunset" → 따뜻한 톤
- "early morning fog" → 차분하고 미스터리한 톤
- "high-key studio" → 밝고 정돈된 톤

각 변형 4컷씩 × 3가지 = 12컷. 이 중에서 클라이언트에 보여줄 3안 큐레이션.

### 4단계: Figma에 배치 (2분)

3안 × 4컷 = 12개 이미지를 Figma 무드보드 템플릿(`Asset > Moodboard Frame`)에
드래그. 각 안에 키워드 라벨링.

## 결과 평가

| 항목                   | 평가                           |
| :--------------------- | :----------------------------- |
| 톤 일관성              | ⭐⭐⭐⭐ (시드 고정 시)        |
| 인물 묘사              | ⭐⭐⭐ (손가락 결함 여전)      |
| 한국적 맥락 표현       | ⭐⭐ (서양 오피스로 기본 편향) |
| 클라이언트 설명 용이성 | ⭐⭐⭐⭐⭐                     |
| 시간 효율              | ⭐⭐⭐⭐⭐ (2h → 15min)        |

## 결정적 팁

1. **`--s` (stylize) 값을 250 정도로 유지** — 기본값(100)은 너무 사실적,
   1000은 너무 일러스트레이션. 250이 무드보드용 절충점.
2. **시드 고정 후 키워드 변형**. 시드 안 잡고 다시 돌리면 톤이 완전히 다른 4컷이
   나와서 톤 일관성이 무너집니다.
3. **한국적 맥락이 필요하면 reference image**. Midjourney v7의 `--cref`로
   한국 오피스/카페 사진을 reference로 주면 동양인 + 한국 컨텍스트가 들어갑니다.
4. **사람 손이 보이는 컷은 reject 기본 전제**. 무드보드에서는 손 안 보이게
   크롭 또는 다른 컷 선택.

## 실패 사례

- "한국 디자이너 협업"으로 직접 프롬프트 → Midjourney가 "한국"을 사극이나
  한복 코드로 해석. 현대 SaaS와 충돌.
- "회의실에서 토론하는 사람들" → 정장 차림 비즈니스맨 일러스트. 톤 부적합.
- `--v 6.1` 사용 → 인물이 v7보다 자연스러우나 톤 일관성은 떨어짐.

## 멤버 실습 과제

본인이 작업했던 최근 프로젝트 1개의 브랜드 키워드를 추출하고, 15분 안에
3안 무드보드를 만들어보세요. **시드 고정 워크플로우를 반드시 적용**해주세요.
일요일 라이브에서 시드/프롬프트/결과를 함께 공유합니다.

## 자주 묻는 질문

- **Q**: 상업적 사용 가능?
- **A**: Midjourney 유료 플랜(Basic 이상)은 가능. 단, 클라이언트 미팅에서
  "이 무드보드는 AI 생성"이라고 명시 권장. 후속 작업에서 동일 이미지 그대로
  쓰기보다는 방향성 참고용으로.

- **Q**: Pinterest 무드보드와 어느 쪽이 좋을까?
- **A**: 둘 다 씁니다. Midjourney = 톤 일관성, Pinterest = 실제 사례 검증.
  병행이 가장 강력합니다.
